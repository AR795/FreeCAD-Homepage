CREATE 3D FROM 2D & BACK

FreeCAD allows you to sketch geometry constrained 2D shapes and use them as a base to build other objects. It contains many components to adjust dimensions or extract design details from 3D models to create high quality production ready drawings.

AND MANY MORE GREAT FEATURES

FreeCAD equips you with all the right tools for your needs. You get modern Finite Element Analysis (FEA) tools, experimental CFD, BIM, Geodata workbenches, a robot simulation module that allows you to study robot movements and many more features. FreeCAD really is a Swiss Army knife of general-purpose engineering toolkits.
